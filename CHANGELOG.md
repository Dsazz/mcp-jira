# Changelog

All notable changes to the JIRA MCP Server project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.2.0] - 2024-03-28

### Added

- New core directory structure with improved organization
- Enhanced error handling system
- Improved logging infrastructure
- Standardized response formatting

### Changed

- Refactored JIRA feature implementation with cleaner architecture
- Reorganized mock structures according to standards
- Simplified test utilities for Bun testing framework
- Enhanced tool handler implementation
- Improved API client structure

### Removed

- Legacy shared directory and components
- Outdated system-time implementation
- Unnecessary mock files

## [1.1.0] - 2024-03-27

### Added

- Migration to Bun runtime and package manager
- Integration with Biome for code formatting and linting

### Changed

- Improved documentation with separate CONTRIBUTING.md

### Removed

- Node.js and ESLint configuration files
